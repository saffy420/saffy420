## 📌 Some projects I'm proud of

### [Docker Firewall Dashboard](https://github.com/saffy420/vlab-firewall)
A Flask dashboard for managing iptables rules on labeled Docker containers in real time, built during my security internship. Custom iptables chains, Docker SDK integration, full audit logging, deployed as a systemd service.

### [Imposter: SAT Vocab Edition](https://github.com/saffy420/imposter-sat-vocab)
A real-time multiplayer vocab game built for my SAT Prep Club after I got tired of flashcards not working. WebSockets, React, Node.js. Members' average Reading & Writing scores went up 20+ points after using it.

### [Trade Tracker](https://github.com/saffy420/trade-tracker)
Cross-platform desktop app (Windows/macOS/Linux) for automating repetitive multi-step workflows, with OCR-based screen reading and platform-specific input automation behind shared interfaces. Built with Avalonia UI, .NET 8, and MVVM.

### [Best Price Checker](https://github.com/saffy420/best-price-checker)
A concurrent web scraper comparing prices across Amazon, Best Buy, Walmart, and Newegg using `ThreadPoolExecutor`. Built for AP CSP's Create Task.

### [Three Selves, One Species: The Amur Leopard](https://github.com/saffy420/amur-leopard)
An interactive web exhibit on phenotypic variation and conservation genetics, built for AP Biology. Drag-to-compare panels, custom hotspots, hand-written CSS layout system.
